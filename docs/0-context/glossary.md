# Glossary

## Domain Terms

| 용어 (Term) | 설명 (Definition) | 동의어/유사어 |
| :--- | :--- | :--- |
| **VR 자위 (VR Masturbation)** | (DLC 전용) 주인공이 집 안의 공포에서 도피하기 위해 매일 밤 수행하는 행위이자 핵심 게임플레이 루프. 귀신과의 접점을 만드는 매개체. | 도피행위 |
| **양기 (Vitality / Yang Energy)** | 주인공의 정신적/영적 생명력. 귀신에게 빨릴수록 감소하며 고갈 시 배드 엔딩 직결. 운동을 통해 전체 최대치를 늘릴 수 있음. | 영적 에너지 |
| **체력 (Stamina/Health)** | 행동력(Action Points)의 기반 수치. 시뮬레이션 행동(일, 운동 등) 시 소모되며 제한되어 있음. 고갈 시 과로로 쓰러지며 직장 생활에 지장을 줌. | 스태미나 |
| **즉석 요리 (Instant Food)** | 요리를 못하는 주인공의 유일한 식사 수단. 인터넷으로 구매하며, 양기와 체력을 회복하기 위한 필수 소모품. | 배달음식, 간편식 |
| **운동 (Exercise)** | 현재의 체력과 양기를 일시적으로 소모하지만, 양기의 전체 최대 한도(Max Cap)를 높여주는 장기 생존 필수 액션. | 홈트레이닝 |
| **직장/알바 (Job/Part-time)** | 즉석 요리를 사기 위해 돈을 버는 수단. 체력 부족으로 못 나가면 수습기간 탈락 후 수익이 적은 '알바' 상태로 강등됨. | 출근 |
| **제물 (Offering)** | 3막에서 귀신의 호감도를 올리거나 사연에 대한 힌트를 얻기 위해 플레이어가 상호작용하는 아이템들. | 조공 |
| **보름달 (Full Moon)** | 귀신의 자제력이 가장 약해지는 주기적 이벤트 타이밍. 스토리 분기와 결전의 핵심 트리거. | 결전일 |
| **정신력 (Sanity / Mental)** | 멘탈 및 공포 저항 수치. 귀신을 보거나 양기 고갈 시 깎이며 0이 되면 귀신에게 빙의당해 자살. 수치가 높을 수록 운동 효율(지속력)이 좋아짐. | 멘탈, 공포도 |
| **생존 일수 (Survival Days)** | 게임의 기본 스코어링 시스템. 정규직 유지 및 효율적 자원 관리를 통해 얼마나 오래 살아남았는지가 기록됨. | 스코어 |

## Abbreviations

| 약어 (Abbr.) | 원어 (Full Terms) | 설명 |
| :--- | :--- | :--- |
| **VR** | Virtual Reality | 주인공이 공포를 잊기 위해 장착하는 가상현실 기기. |
| **VN** | Visual Novel | 대화 및 스토리텔링 연출 방식 (예: 4막 귀신의 회상 씬). |
