## 장르 — 생활 시뮬레이션 + VN 연출

플레이어가 **일상 활동을 직접 관리**하면서 스토리가 진행됨.
핵심 장면에서 **VN 스타일 연출** (대사, 회상, 선택지)과
**아케이드성 인터랙션**, **성감 시뮬레이션**이 삽입.

### 일상 활동 (사용자 관리)

| 활동 | 설명 |
|------|------|
| **식사** | 체력/양기 유지에 영향 |
| **세신** | 위생/컨디션 관리 |
| **청소** | 집 상태 관리 (귀신이 도와주기도) |
| **휴식** | 체력 회복, VR 사용 시간 |
| **일** | 게임 회사 출근, 수입 확보 |
| **쇼핑** | 식량, 생활용품, VR 콘텐츠 등 구매 |

### 공포 인터랙션 (아케이드)

| 상황 | 조작 | 실패 시 |
|------|------|---------|
| **가위눌림** | 줄어드는 게이지를 **키 연타**로 채움 (고개를 돌리려는 발악) | 게이지 소진 → 공포 누적 |
| **인기척** | **좌/우 키**로 균형 조절 — 돌리고 싶은 마음 vs 돌리고 싶지 않은 마음 | 너무 일찍 돌림 → 귀신 목격 → 놀라서 도망 → 퇴마 루트 |
| **젖 빨기 (자는 척)** | 잠든 것처럼 **자연스러운 리듬**으로 키 입력 (혀 움직임) | 리듬 깨짐/너무 급함 → 귀신이 눈치 챔 → 사라짐 |

> **젖 빨기 리듬**은 4막 **성감 개발의 리듬 터치**와 같은 메커니즘의 초기 버전.
> 플레이어가 자연스럽게 리듬 감각을 익히는 훈련 역할.

### 성감 시뮬레이션

- 젖꼭지, 가슴, 성기 등을 **사용자가 직접 조작** (터치/클릭)
- **성감을 느끼기 좋은 곳**을 찾아 **적당한 리듬감**으로 터치하는 것이 핵심
  - 빠른 액션이 아닌, **느리고 적절한 리듬**이 중요
  - 올바른 부위 + 올바른 리듬 = 흥분 상승
  - 잘못된 부위 or 너무 빠름/느림 = 반응 약함
- 반복적 행위는 **미리 설정된 오토 기능**으로 전환 가능
- **숨겨진 흥분도** — 별도의 게이지 UI를 표시하지 않음
- 렌더링 방식 (Live2D / 3D 셀 셰이딩 / CG 파츠 교체)은 **추후 결정**

#### 시각적 흥분 피드백 (게이지 대체)

| 단계 | 시각적 변화 |
|------|-----------|
| 초기 | 표정 변화, 가슴이 부풀어 오름 |
| 중반 | 젖꼭지가 섬, 호흡 거칠어짐 |
| 후반 | 클리토리스·소음순 부풀어오름, 애액 증가 |
| 절정 | 몸 경련, 표정 변화, 절정 반응 |

> **핵심**: 플레이어가 **몸의 반응을 관찰**하며 흥분도를 파악해야 함.
> 숫자 게이지가 아닌 **시각적 변화**로만 상태를 알 수 있음.

> [!IMPORTANT]
> **디자인 원칙: 수치 비노출**
> 내부적으로 흥분도·공포도·양기 등의 수치가 존재하더라도,
> 플레이어에게는 **어떤 수치도 직접 보여주지 않는다**.
> 모든 피드백은 **시각적 변화와 연출**로만 전달.

## 주제의식

> **이승의 존재와 저승의 존재는 함께할 수 없다.**

귀신이 아름다워질수록 주인공은 죽어간다 — 한쪽이 살면 한쪽이 소멸하는 구조.
이 메시지는 대사가 아닌 **캐릭터 외형의 역방향 변화**로 전달된다.

```
귀신:  흉측한 악귀  →→→→→  아름다운 여성
주인공: 통통한 오타쿠  →→→→→  미라 같은 폐인
```

모든 엔딩에서 이 룰은 깨지지 않는다:
- **배드**: 함께했더니 산 자가 죽었다
- **노멀**: 함께할 수 없으니 죽은 자를 내쫓았다
- **트루**: 함께할 수 없으니 죽은 자가 스스로 떠났다 (성불)

## 로그라인

억울하게 죽어 악귀가 된 처녀귀신이 사는 집에 이사 온 내성적인 남자.
남자의 **양기(정액)**가 악귀의 **탁기**를 정화하면서 둘의 관계가 변해간다.

## 주인공

| 항목 | 설명 |
|------|------|
| **성별** | 남성 |
| **나이** | 20~40대 |
| **성격** | **히키코모리 모쏠 오타쿠** — 겁이 많고 유약하며 모질지 못함 |
| **특징** | 귀신이 무서워 VR을 구매 → 공포 도피 + 자위가 일상 |
| **생활습관** | **전자 담배** 사용, **인스턴스/편의점 음식** 주식 → **집에서 불을 전혀 안 씀** |
| **페티쉬** | **모유 페티쉬** — 엄마가 일찍 죽어 젖을 물린 기억이 없음 (편부 가정) |

### 외형

| 부위 | 묘사 |
|------|------|
| **눈** | 처진 눈 + 짙은 다크서클 + 약간 크고 불안한 동공 |
| **피부** | 창백 (햇빛을 안 봄) + 코끝·귀끝만 약간 붉음 + **턱라인 성인 여드름** |
| **머리** | 까맣고 약간 긴 생머리, 정돈 안 됨 (떡진 머리) |
| **체형** | 키 작음, 좁은 어깨, 마른 팔다리 + **복부비만** (운동 제로 + 야식의 결과) |
| **복장** | 늘어난 **검정 후드티** + 트레이닝 바지 (항상 같은 옷) |
| **소품** | **파란 LED 전자담배** — 어두운 방에서 유일하게 빛나는 것 |

### 막별 외형 변화

| 막 | 변화 | 인상 |
|----|------|------|
| **1막** | 통통 + 복부비만 + 여드름 + 다크서클 | 찌질한 히키코모리 |
| **2막** | 거의 동일 (미세하게 야위기 시작) | 정액 소모 시작 |
| **3막** | 뚜렷하게 야윔 + 다크서클 심화 | 매일 밤 정액을 빼앗김 |
| **4막 초반** | 광대뼈 도드라짐, 꺼진 볼, 핏줄 비침 | **미라 같은 얼굴** — 동료가 걱정할 수준 |
| **4막 후반** | 소모 속도 감소, 약간 회복 | 성감 개발 성공 → 생존 안정화 |

> **배경**: 엄마가 일찍 죽고 **아버지 손에서 자람**.
> 히키코모리 생활 중 **아버지의 지원이 끊김**.
> 자신이 익숙한 분야인 **게임 콘텐츠를 만드는 작은 회사**에 입사.
> 수도에 왔지만 돈이 없어, 여성 세입자들이 연이어 자살해 소문이 퍼진 이 집이 **파격적으로 싸게** 나와 입주.

## 처녀귀신 (히로인)

### 과거
- 남편은 **귀한 독자** — 시댁에서는 손주를 간절히 바람
- 여자는 **고아** — 가족이 없었기에 **자신만의 가족을 갖는 것**이 평생의 소원
- 시댁에서 **결혼을 반대** (고아와의 결혼을 탐탁지 않게 여김)
- 제대로 된 결혼식을 올리지 못하고, **야반도주하듯** 따로 살림을 차림
- 둘이서 **소박한 예물로 약조** — 화려하진 않지만 진심을 담은 서약
- **혼인신고서도 제출하지 못함** — 법적으로는 부부가 아닌 채로 죽음
- 집이 불에 타면서 **둘의 관계를 증명할 모든 것이 소실** — 세상 사람들은 둘의 사이를 알지 못함
- 남편의 아이를 갖는 것은 시댁의 기대만이 아닌, **자신의 진심이기도 했음**
- **초야**를 치르기 위해 집에서 준비하고 있었음
  - 목욕을 하고 속옷을 갈아입고
  - 침대를 정리하고 향초를 켜고
  - 설렘과 긴장 속에 남편을 기다리는 중
- 같은 밤:
  - 남편: 집으로 오는 길에 **교통사고**로 사망 → 사고 현장 부근에 **남자 귀신**으로 남아있음
  - 여자: **방화범이 지른 불**에 의해 집에서 **질식사**
- 사랑하는 남자의 씨를 **단 한 번도 받지 못한 채** 죽음
- 초야를 치르지 못했기에 **성교하는 법을 모름** — 본능적 욕구만 있을 뿐
- 남편의 귀신도 아내에게 가지 못한 한을 품고 사고 현장에 머물고 있음
- 사고 현장에는 **시어머니(노녀)**가 종종 꽃다발을 들고 찾아와 슬퍼함
- 사후 집터는 **여자 전용 기숙사/임대 건물**로 운영됨 (화재로 재건축 후)
- 남자가 없는 환경에서 한이 점점 쌓임
- 급기야 여자 입주자들을 **반 강제로 자살**시킴 → **악귀**로 변질
- 방화범이 살인 현장을 추억하기 위해 찾아왔다가 **귀신에게 죽음**
- 하지만 살해한 당사자를 죽이고도 **성불하지 못함** → 원한이 오히려 증폭
- **화기(불) 트라우마**: 방화로 죽었기에 불을 극도로 무서워함
  - 주인공의 평소 생활(전담, 인스턴스) 덕에 평소에는 발현되지 않음
  - 하지만 **향초나 향(제물)**을 피우려 하면 발작적인 거부 반응을 보임 → 사연의 단서

> **한의 본질**: 고아로서 가족이 없었기에, **자신만의 가족을 가지는 것** — 엄마가 되는 것
> — 인생에서 가장 행복했어야 할 밤에 **모든 것을 빼앗겼다**
> (시댁의 대를 잇는 것은 시댁의 기대일 뿐, 귀신 자신에게는 중요하지 않음)
>
> **사연의 단서 표현**:
> - 집 안의 유품: 오래된 족보, 시어머니의 편지 ("꼭 손주를 봐야 한다"), 아기 옷 선물
> - 남편 귀신의 방황: "집안의 대를 이어야 한다는 게... 아내에게 미안했다" (귀신 회상으로 전달)
> - 여자 귀신의 회상: 시댁의 압박, 그래도 설레던 마음

### 악귀 상태 — 외형

거의 인간의 모습이 아니다.

| 부위 | 묘사 |
|------|------|
| **하반신** | 두 다리는 지워져 없음. **배꼽 아래는 흐릿하게 소멸** |
| **얼굴** | **해골처럼 눈이 휑** — 동공이 없는 구멍. 코는 흔적만 남음 |
| **입** | **잇몸이 없어서 이빨이 더 크고 뻣뻣하게** 보임 |
| **팔** | **네 개의 팔** — 몇 명의 여성을 자살하게 만든 팔. 뼈만 남은 듯 앙상하지만 **강인하고 억세게** 보임 |
| **전체** | 비물질적, 떠다님, 흐릿한 윗벳 같은 존재 |

- 혼자 사는 사람을 쇠약하게 만들어 자살로 몰아감

### 정화 과정 (= 잊었던 자신을 기억하는 과정)

정액의 양만큼 복원되는 것이 아니라, 정액이 **잊었던 자신의 모습을 떠올리게 하는 촉매** 역할을 한다.

| 단계 | 계기 | 변화 | 접촉 |
|------|------|------|------|
| **악귀 상태** | — | 흉측, 하반신 없음, 해골 같은 얼굴 | 비물질 |
| **1차 정액 노출** | 배꼽 아래에 정액이 닿음 | **배꼽 주변에 온기** — 잊었던 무언가(아이를 갖고 싶던 마음)를 떠올림 | 보이지만 못 만짐 |
| **반복 노출** | 점점 더 많은 정액을 맞으려 함 | **가슴, 눈을 되찾음** — 눈을 되찾은 순간 **눈물을 흘림** | 귀신이 남자를 만질 수 있음 |
| **구강 성교** | 적극적으로 정액을 받으려 함 | 아름다운 여성으로 완전 복원 | **서로** 만질 수 있음 |

> 정액은 **양적 연료**가 아니라 **기억의 촉매** — 맞을수록 생전의 자신을 더 많이 기억해내고, 그만큼 더 간절히 원하게 된다.

### 원래 외모 (생전 / 완전 복원 시)

| 부위 | 묘사 | 서사적 의미 |
|------|------|----------|
| **키** | 아담한 편 (크지 않음) | 고아라 성장기에 잘 먹지 못했음 |
| **어깨** | 좁은 어깨, **라운드 숄더** | 큰 가슴을 숨기려는 습관에서 비롯된 체형 |
| **가슴** | **풍만함** | 모성애의 상징. 젖 물리기와 직결 |
| **골반·엉덩이** | **풍만하게 발달** | 아이를 갖고 싶어하는 마음의 신체적 표현 |
| **아랫배** | **젖살이 덜 빠진 귀여운 느낌** | 어린 느낌과 여성스러움의 공존 |
| **전체 체형** | 마른 편은 아님 — 어른이 되어 먹는 걸 참지 않았음. 통통하진 않지만 가슴과 엉덩이가 풍하게 보이는 체형 | 결핍에서 벗어나 삶을 누리려는 의지 |
| **성격** | 남편에게 의존적, 순종적. 시댁에 맞서는 강단은 없음 | 남편이 사라지고 홀로 남았을 때 무너지는 이유 |

### 얼굴 가시성 변화

| 관계 단계 | 얼굴 | 대화 |
|----------|------|------|
| 초기 성접촉 | **흐릿함** — 입술·가슴·성기만 뚜렷 | 불가 |
| 관계 심화 | 점점 **선명해짐** | 단편적 반응 |
| 깊은 유대 | **완전히 보임** | **대화 가능** → 사연을 듣게 됨 |

### 이름의 복원
- 초기에는 이름이 없음 ("귀신", "그녀")
- 유품(혼인신고서 초안 등) 조사를 통해 **이름을 알아내고 불러주는 과정** 필수
- 이름을 불러주는 행위는 귀신의 자아 복원과 성불에 결정적 역할

### 귀신의 호의 (관계 심화 후)
- 집 **청소**를 해줌
- 물건이 쓰러질 때 **맞지 않게** 보호해줌
- 점점 가정적인 행동 (악귀 → 아내/엄마의 본능 회복)

> **젖 물리기는 1막부터 귀신이 몰래 해왔음** (1막: 몰래, 2막: 반쯤 인지, 3막: 익숙해져서 만지면서 빨음)
> 주인공이 의식적으로 인지하는 것은 **젖이 나오기 시작한 이후** (4막)

---

## 핵심 메커니즘

### 양기와 탁기

```
남자의 정액 (양기/생명의 씨)
    ↓
악귀의 탁기를 정화
    ↓
귀신의 모습·감각·자아 복원
```

### 균형 시스템

| 상태 | 결과 |
|------|------|
| 적당한 사정 | 귀신 만족 → 정신을 차리고 모습을 감춤 |
| 과도한 사정 | 남자의 양기 고갈 → **사망** |
| 오랫동안 정액 미제공 | 귀신이 발정 → **과격한 성교** → 복상사 위험 |

### 성감 개발

- 첫 성교 후 귀신이 **과도하게 사정을 요구** → 주인공이 체력 저하/위기를 느낌
- "이러다 죽겠다" → **사정 없이 만족시키는 방법**을 깨달아야 함
- 귀신도 초야를 치르지 못한 처녀라 **성감을 모름** → 함께 배워가는 구조
- 성감 개발이 진행될수록 → 정액 집착 감소 → **생존 안정화**
- → 양기 고갈/복상사 위험을 낮추는 **핵심 생존 메커니즘**

## 스토리 흐름 (기승전결)

> 상세는 [scenario.md](scenario.md) 참조

### 1막: 자위 (기)
- 이사, 공포, VR 도피 → 자위가 일상이 됨
- 악귀가 잠든 주인공에게 **젖 물리기 시작** (플레이어만 봄)

### 2막: 귀신의 변화 (승)
- 정액 노출 → 악귀 복원 (주인공 모름, 플레이어만 봄)
- 귀신이 VR 중 성기를 빨다가 발각 → 숨음
- 점점 거리 좁혀짐 (팜플렛, 젖 물리기 반복)

### 3막: 유혹 (전)
- 주인공이 귀신을 불러내려 시도 (제물, 검색)
- 보름달에 귀신 발정 → **첫 성교** (해 떴 때까지 질내사정)
- 이후 성적 관계 일상화 → 귀신이 모습을 드러내고 기다림

### 4막: 성감 개발 (결)
- 귀신의 성욕 에스컬레이션 → 다크서클, 양기 부족 → **성감 개발** 시작
- 젖이 나옴 → **모유 페티쉬** → 귀신이 아기를 떠올림
- 얼굴 선명 → 대화 가능 → 사연 해명 (VN 회상)
- 출장 귀환길에 노녀 구출 → 남편 귀신 성불 → 아내의 거부감 해소
- **두 번째 보름달** → 엔딩 분기

## 멀티 엔딩

### 배드 엔딩

| 엔딩 | 조건 | 결과 |
|------|------|------|
| **심장마비** | 공포 이벤트에서 과도한 공포 | 남자, 놀라서 심장마비로 사망 |
| **복상사** | 오랫동안 정액을 주지 않음 (동료의 부적 등) | 귀신 성욕 폭발 → 과격한 성교 → 정기를 빨려 남자 사망 |
| **악귀 회귀** | 귀신이 나온 집이 무서워 이사 나감 | 집은 다시 악귀의 집으로... (순환) |
| **양기 고갈** | 동거 엔딩 후 | 귀신이 성불을 포기하고 동거 → 남자의 양기가 말라 사망 |

### 노멀 엔딩

| 엔딩 | 조건 | 결과 |
|------|------|------|
| **퇴마 (부적)** | 동료가 부적 선물 | 귀신이 한동안 나타나지 못함 → 성욕 폭발 → 분기 |
| **퇴마 (퇴마사)** | 남자가 퇴마사를 부름 | 귀신 퇴치 |
| **퇴마 (상담)** | 누군가에게 상담 → "전형적인 처녀귀신" 진단 | 귀신을 죽이는 부적을 받아 퇴치 |

### 트루 엔딩

**필수 조건**: 주인공이 노녀를 구함 → 남편 귀신이 따라와 성불 → 아내 귀신의 한 충족

| 엔딩 | 조건 | 결과 |
|------|------|------|
| **성불** | 남편 귀신 성불 + 보름달 성교 + 한 충족 | 귀신이 **임신**한 것을 알게 됨 (젖이 나옴, 입덧) → 성불 |

> 주인공이 노녀(시어머니)를 구한 선행 → 남편 귀신이 집까지 따라옴
> 남편 귀신이 아내 귀신을 보고 성불 → 아내에게 전하지 못한 마음이 전해짐
> 주인공은 이 과정을 모름 — 자신의 선행이 한 해소의 열쇠였음을 알지 못함
>
> **성불의 흐름**:
> 1. 보름달 밤에 성교 → 임신
> 2. 여자 귀신이 **입덧**을 시작 → 임신을 확신하게 됨
> 3. 남자가 귀신의 가슴을 빨는데 **젖이 나옴** (이번엔 진짜 모유)
> 4. 귀신이 자신의 뱃속에 새 생명이 깃든 것을 느끼며 —
> 5. 고아였던 자신이 엄마가 된다는 것을 알게 되며 — 한이 풀리고 성불.
