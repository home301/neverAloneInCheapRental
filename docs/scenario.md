## 기승전결 4막 구조

| 막 | 이름 | 핵심 | 플레이 느낌 |
|----|------|------|-----------|
| **기** | 자위 | 이사, 공포, VR 도피 | 호러 + 생활 시뮬 |
| **승** | 귀신의 변화 | 정액→정화, 악귀 복원 (주인공 모름, 플레이어만 봄) | 호러 + 관음적 긴장 |
| **전** | 유혹 | 귀신을 불러내려는 시도, 제물 시스템 | 탐색 + 분기 집중 |
| **결** | 성감 개발 | 관계, 호의, 사연, 엔딩 | 풀 시뮬 + 서사 |

## 1막: 자위 (기)

**주인공 상태**: 공포에 질린 히키코모리
**귀신 상태**: 흉측한 악귀, 위협적

### 흐름
- 이사, 정착, 첫 출근
- 밤에 소름끼치는 현상 발생
- 가위눌림 (공포 인터랙션)
- 공포에서 도피하기 위해 **VR 구입 → 자위 시작**
- VR 자위가 일상이 됨
- 주인공이 잠결에 손가락을 빠는 버릇 → 악귀가 **가슴을 물려봄** → 주인공이 맛있게 빠듬 → **매일 밤 젖 물리기 시작** (플레이어만 봄)

### 분기점
- 공포 누적 과다 → **심장마비 엔딩**
- 이사를 결심 → **악귀 회귀 엔딩**

### 메뉴
- 식사, 세신, 출근, 쇼핑, VR 자위, 수면
- **인터넷 검색**: "이 집의 괴담", "과거 방화 사건 뉴스" (사연의 파편 수집 시작)

## 2막: 귀신의 변화 (승)

**주인공 상태**: VR 자위 루틴. 귀신 흔적이 사라져서 안심.
**귀신 상태**: 정액에 노출되며 서서히 복원 중 → 성기에 집착
**핵심**: 플레이어만 귀신의 변화를 봄 — **극적 아이러니**

### 흐름
1. VR 자위 중 악귀 접근 → 사정 시 정액이 묻음
2. 악귀의 아랫배 복원 → 놀라서 달아남 (**플레이어만 목격**)
3. **어느 순간부터 귀신의 흔적이 사라짐** — 주인공 안심
4. VR 자위가 **점점 실감나게** 느껴짐 — 주인공은 VR 성능 덕이라 생각
   - **실상**: 귀신이 VR 중인 남자의 **성기를 빨아주고 있음** (플레이어만 봄)
5. 정액에 반복 노출 → 귀신의 모습이 점점 복원 (**플레이어만 봄**)
6. 귀신, 성기에 집중하느라 **남자가 VR을 벗는 걸 알아채지 못함**
7. **남자, VR을 벗음 → 여자가 성기를 빨고 있는 장면 목격**
8. 귀신, 들킨 것을 알아채고 **급히 흐릿하게 모습을 감춤**
9. 남자, 흐릿한 형체에 **놀람** — "지금 뭐가 있었나?"
10. 구강 성교의 촉감이 **기억에 남음**

#### 발각 이후 — 귀신과의 거리 좁히기

| 단계 | 귀신의 행동 | 주인공의 반응 |
|------|-----------|-------------|
| **첫 발각 직후** | 흐릿하게 사라짐. 한동안 모습 안 보임 | "뭐였지?" 혼란. 촉감만 기억에 남음 |
| **며칠 후** | 구석에 앉아 조용히 지켜봄 (종종 모습을 드러냄) | 인기척에 고개 돌리면 → 여자가 앉아있는 걸 봄 |
| **다가가면** | 도망감. 벽에 숨으려는데 **상체만 숨고 하체가 삐져나옴** (실체화로 벽 통과 불가) | "저건 뭐지?" 다가감 |
| **만지면** | 촉감 있음 → 하지만 **즉시 사라짐** | "만져졌는데 사라졌다 = 귀신이다" → **확신** |
| **젖 물리기 (계속)** | 1막부터 이어져 온 매일 밤 젖 물리기 반복 중. 가끔 잠이 깨지만 **자는 척하면서 빠야** 함 (들키면 사라지니까) | 주인공은 아직 자각 없음 — 젖이 나오기 전까지는 모름 |
| **팜플렛 사건** | 회사에서 젊은 총각 대상 결혼 이벤트로 받아온 **결혼 장려·출산 권장 팜플렛**을 귀신이 빤히 쳐다보다 들킴 | "이 귀신, 이런 거에 관심이 있나?" → 3막 제물 힌트 |
| **반복** | 점점 도망 안 치고 **가까이서 지켜봄** | "나와봐" → 3막 유혹 시도로 |

### 분기점
- 동료가 안색 인지 → (3막 이벤트 준비)

### 메뉴
- 1막과 동일 (변화 없음 — 주인공은 모르니까)

## 3막: 유혹 (전)

**주인공 상태**: 귀신을 확신. 다시 만지고 싶다. 불러내고 싶다.
**귀신 상태**: 들키고 부끄러워 숨지만, 관심은 있음. 정면으로는 나타나지 않음.
**핵심**: 귀신을 다시 나타나게 하기 위한 **플레이어의 능동적 시도**

### 흐름
- 구강 성교 후 주인공은 촉감이 있었던 것에 놀람
- 다시 시도하지만 **귀신이 나타나지 않음**
- VR 자위를 해도 반응 없음
- 주인공, 인터넷 검색으로 귀신을 부르는 방법 시도

#### 밤의 비밀 (주인공 반쯤 인지)
- 3막에선 젖 먹이기에 **이미 익숙해짐** — 주인공이 **가슴을 만지면서 빨음**
- 귀신의 밤 루틴: 젖을 물림 → 자위를 해줌 → **정액을 받아 마심**
- 주인공은 반쯤 깨어있지만 편하게 받아들이는 상태

> **보름달이 가까워질수록** 귀신의 자제력이 약해짐:
> - 초반: 주인공이 눈뜨기 전에 사라짐
> - 중반: 주인공이 눈을 떴는데 **아직 성기를 빨고 있다가** 뒤늦게 사라짐
> - 후반: 사라지는 속도가 느려짐, **잠깐 눈이 마주침**
> - **보름달 당일**: 발정으로 사라지지 않음 → 남자의 몸을 핥고 빠는 귀신

#### 첫 보름달 밤 — 첫 성교
- 귀신이 발정으로 남자의 몸을 **핥고 빠는** 데서 시작
- 남자가 여자의 성기를 빨고 → **삽입** → 사정하려고 빼려 하지만
- 귀신이 **두 다리를 허리에 감고** 질싸를 유도하며 **"임신 시켜달라"**고 외침
- 주인공 놀람 (아직 대화 불가 — 이 한마디만 기억에 남음)
- 해가 뜰 때까지 질내 사정을 반복
- 귀신은 그 **쾌감과 만족을 잊지 못하게 됨** → 4막으로

#### 첫 성교 이후
- 귀신이 **다시 질싸를 받고 싶어짐** → 주인공이 자는 사이 성교를 시도
- 하지만 잠든 상태에서는 **삽입이 잘 안 됨**
- 결국 주인공이 **깨서 성교** → 또 여러 차례 사정
- 이 패턴이 반복되며 **성적 관계가 일상화**
- 이후에는 귀신이 집에 오면 **성교를 하려고 모습을 드러내고 있음** → 4막 돌입

### 제물/시도 시스템

#### 반응 없음 ❌

| 시도 | 결과 |
|------|------|
| 고기, 술 (전통 제물) | 변화 없음 |
| 자위기구 | 변화 없음 |
| 속옷 | 변화 없음 |

#### 반응 있음 ✅ (귀신의 한과 관련)

| 시도 | 귀신 반응 | 효과 |
|------|----------|------|
| **화목한 가정 사진** | 물끄러미 바라봄 | 호감↑ 성욕↑ |
| **아기 인형/용품** | 손을 뻗으려 함 | 호감↑ 성욕↑↑ |
| **소박한 반지** | 강하게 반응 | 호감↑↑ |
| **꽃/향초** | **초기엔 가까이 옴 / 불을 붙이면 발작** | 사연 힌트 (불 트라우마) |

> **불 트라우마**: 주인공이 불을 안 쓰기에 평소엔 모르다가, 제물용 향을 피울 때 귀신이 격렬히 저항하며 발각.

> 플레이어가 **"이 귀신이 진짜 원하는 것"**을 유추하는 게임플레이.
> 성적인 것에는 무반응 — 한의 본질은 가정·아이·사랑.

### 분기점
- 동료의 부적 선물 → 귀신 차단 → 보름달에 성욕 폭발 → **복상사 엔딩**
- 상담 → "전형적 처녀귀신" 진단 → 퇴마 부적 → **퇴마 (상담) 엔딩**
- 퇴마사 호출 → **퇴마 (퇴마사) 엔딩**

### 전환 조건
- 호감·성욕 수치가 충족되면 → **보름달 밤에 귀신이 나타나 성교**
- 첫 성교 후 반복 → 4막으로 이행

### 메뉴 추가
- **제물 놓기**, **인터넷 검색**, **주변 대화**

## 4막: 성감 개발 (결)

**주인공 상태**: 귀신과 성적 관계. 점점 유대가 깊어짐.
**귀신 상태**: 정기적으로 나타남. 호의적. 얼굴 점점 선명.
**핵심**: 성감 시뮬레이션, 사연 해명, 엔딩 분기

### 흐름
- 보름달 밤 성교 후 **섹파처럼 지내기로** 관계 성립
- 귀신의 성욕이 점점 에스컬레이션:
  - 옷도 안 벗었는데 **바지 내리고 팬티 내리고 성기를 꺼냄**
  - **공중에서 삽입**해버리기까지 함
- 주인공은 **다크서클이 심해지고 양기가 부족해짐** → 체력 한계
- **인터넷 검색** + **지인에게 상담**: "여자가 너무 수가 많아서 힘들다"
- 조언: **"여자의 성감을 개발해서 먼저 절정에 이르게 해라"**
- 둘 다 처음 (모쏠 × 처녀귀신) → **함께 배워가는 성감 개발**
- 귀신의 호의 시작 (청소, 보호, 가슴 물려줌)
- 얼굴 흐릿 → 선명 → **대화 가능**
- 첫 성교 때의 **"임신 시켜달라"를 기억**하고 물어봄 → 사연을 듣게 됨 (VN 연출/회상) — **방화범 정보도 이 과정에서 전달**
- 유품 발견 (족보, 편지, 아기 옷, 반지)
- **이름 찾기**: 유품 조사 중 혼인신고서 초안 발견 → **이름을 불러줌** (자아 복원)
- 젖이 나오기 시작 → 주인공이 **처음으로 젖 물리기를 인지** ("이게 무슨 맛이지?")
- 젖이 나오는 것을 알고 **의식적으로 모유 페티쉬**로 발전
- 귀신이 젖 빠는 남자를 보고 **아기를 떠올림** → 임신 욕구
- 주인공이 **출장에서 일찍 돌아오는 길에** 사고 현장에서 노녀(시어머니)가 투신하려는 것을 말림
- 남편 귀신이 선행에 이끌려 **집까지 따라옴** → 아내 귀신을 보고 **성불**
- 아내 귀신은 남편의 성불을 지켜봄 → **주인공에 대한 거부감 해소**
- (주인공은 이 과정을 모름 — 귀신과 플레이어만 봄)
- **두 번째 보름달** → 엔딩 분기

### 엔딩 분기

| 조건 | 엔딩 |
|------|------|
| 남편 귀신 성불 + 보름달 성교 | **트루 엔딩 (성불)** — 입덧, 진짜 모유, 한 해소 |
| 남편 성불 미완 + 보름달 성교 | **양기 고갈 엔딩** — 임신하지만 성불 실패, 동거, 양기 소진 |
| 동거 선택 | **양기 고갈 엔딩** |

### 메뉴 추가
- **성감 개발**, **대화**, **유품 조사**, **외출 조사 (사고 현장)**

## 일상 시뮬레이션 루프

```
[기상] → [세신] → [식사] → [출근]
                                │
                           [회사 이벤트]
                                │
[퇴근] → [쇼핑?] → [식사] → [청소?]
                                │
                           [밤 자유 시간]
                        ┌───┼───┐───┐
                      [VR] [귀신] [조사] [제물]
                                │
                          [수면/이벤트]
```

### 행동과 수치의 관계 (내부, 비노출)

| 행동 | 영향 |
|------|------|
| 식사 | 체력↑, 양기 회복 보조 |
| 세신 | 컨디션↑ |
| 청소 | 집 상태↑ |
| 휴식 | 체력↑, 공포↓ |
| 일 | 수입↑, 동료 이벤트 |
| VR 자위 | 공포↓, 양기↓ |
| 귀신 접촉 | 관계↑, 양기↓ |
| 제물 | 귀신 호감/성욕 변동 |
| 조사 | 정보 수집 → 스토리 진행 |

## 엔딩 요약

| 엔딩 | 타입 | 발생 시점 | 조건 |
|------|------|----------|------|
| 심장마비 | 배드 | 1~2막 | 공포 임계 초과 |
| 악귀 회귀 | 배드 | 1~2막 | 이사 선택 |
| 복상사 | 배드 | 3막 | 부적 → 귀신 차단 → 보름달 성욕 폭발 |
| 퇴마 (퇴마사) | 노멀 | 3막 | 퇴마사 호출 |
| 퇴마 (상담) | 노멀 | 3막 | 상담 → 퇴마 부적 |
| 양기 고갈 | 배드 | 4막 | 동거 or 남편 성불 실패 |
| **성불** | **트루** | **4막** | **남편 성불 + 보름달 성교 + 한 충족** |
