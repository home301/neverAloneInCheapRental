| # | 씬 이름 | 타입 | 설명 |
|---|---------|------|------|

| 2-02 | 정액 묻음 → 아랫배 복원 | VN | 사정 시 정액이 묻어 아랫배(자궁) 복원 → 악귀 놀라 도주 |
| 2-03 | 귀신 흔적 사라짐 | VN | 주인공이 안심하는 일상 — 실상은 귀신이 은밀히 접근 중 |
| 2-04 | VR이 실감나게 느껴짐 | VN | "VR 성능 좋아졌나?" — 실상: 귀신이 성기를 빨아줌 (플레이어만 봄) |
| 2-05 | 귀신 모습 점진적 복원 (반복) | VN | 정액 반복 노출 → 인간 여성으로 복원 (플레이어만 봄) |
| 2-06 | **발각** — VR 벗음 | VN | 남자가 VR을 벗음 → 여자가 성기를 빨고 있는 장면 목격 |
| 2-07 | 귀신 사라짐 후 혼란 | VN | "뭐였지?" — 촉감만 기억에 남음 |
| 2-08 | 며칠 후 — 구석의 여자 | VN | 귀신이 구석에 앉아 조용히 지켜봄. 고개 돌리면 목격 |
| 2-09 | 벽에 숨는 귀신 | VN | 도망치다 상체만 숨고 하체가 삐져나옴 (실체화로 벽 통과 불가) |
| 2-10 | 만지면 사라짐 | VN | 촉감 있음 → 즉시 사라짐 → "귀신이다" 확신 |
| 2-11 | 젖 빨기 (자는 척) | ARC | 가끔 잠이 깨지만 자는 척하면서 리듬 맞춰 빠는 아케이드 |

| 2-13 | 점점 가까워지는 귀신 (반복) | VN | 도망 안 치고 가까이서 지켜봄 → "나와봐" |
| 2-14 | 동료가 안색 인지 | VN | 회사에서 동료가 주인공의 안색을 걱정 (3막 부적 이벤트 복선) |

| # | 씬 이름 | 타입 | 설명 |
|---|---------|------|------|

| 3-05 | 제물 놓기 (반응 있음) | SIM/VN | 가정 사진, 아기 인형, 반지, 꽃 → 귀신 반응 |
| 3-06 | 불 트라우마 발견 | VN | 향초에 불을 붙이자 귀신의 발작적 거부 반응 |
| 3-07 | 밤의 비밀 — 젖 빨기 (익숙) | ARC/SEX | 3막에선 가슴을 만지면서 빨음. 귀신의 밤 루틴 |
| 3-08 | 보름달 접근 — 자제력 약화 (반복) | VN | 눈뜨기 전 사라짐 → 빨다가 뒤늦게 사라짐 → 눈 마주침 |
| 3-09 | **첫 보름달 밤 — 첫 성교** | SEX | 발정 → 핥고 빨기 → 삽입 → 빼려 하자 **다리 감기 + "임신 시켜달라"** → 질내사정 반복 |
| 3-10 | 첫 성교 이후 — 삽입 시도 | SEX | 귀신이 잠든 주인공에게 삽입 시도 → 실패 → 깨서 다시 성교 |
| 3-11 | 성적 관계 일상화 | VN | 귀신이 모습을 드러내고 성교를 기다림 → 4막 돌입 |
| 3-E1 | **복상사 엔딩** | END | 동료 부적 → 귀신 차단 → 보름달 성욕 폭발 → 사망 |
| 3-E2 | **퇴마 (상담) 엔딩** | END | 상담 → 처녀귀신 진단 → 퇴마 부적 → 퇴치 |
| 3-E3 | **퇴마 (퇴마사) 엔딩** | END | 퇴마사 호출 → 귀신 퇴치 |

| 4-05 | **성감 개발 (반복)** | SEX | 둘 다 처음 → 함께 배우는 리듬 터치 시뮬레이션 |

| 4-08 | "임신 시켜달라"의 의미 | VN | 첫 성교 때의 말을 기억하고 물어봄 → 사연 듣기 시작 |

| 4-11 | **이름 찾기 & 이름 부르기** | VN | 혼인신고서에서 이름 발견 → 이름을 불러줌 → 자아 복원 결정타 |

| 4-14 | 귀신이 아기를 떠올림 | VN | 젖 빠는 남자를 보고 아기를 떠올림 → 임신 욕구 강화 |
| 4-15 | 출장 | VN | 회사 출장 발생 |
| 4-16 | **노녀 구출** | VN | 출장 귀환길에 사고 현장에서 투신하려는 노녀를 말림 |
| 4-17 | **남편 귀신 성불** | VN | 남편 귀신이 따라옴 → 아내를 보고 깨달음 → 성불 (귀신/플레이어만 봄) |
| 4-18 | 아내의 거부감 해소 | VN | 남편 성불을 지켜본 후 주인공에 대한 감정이 열림 |

| 4-E1 | **트루 엔딩 (성불)** | END | 임신 확인 (입덧, 진짜 모유) → 한이 풀리고 성불 |
| 4-E2 | **양기 고갈 엔딩** | END | 남편 성불 실패 or 동거 선택 → 양기 소진 사망 |

## 필요 배경

| # | 배경 | 사용 빈도 | 비고 |
|---|------|----------|------|
| 1 | **집 — 뷰 A (침대 쪽)** | ★★★★★ | 수면, VR 자위, 성교, 젖 물리기. 밤 씬의 메인 뷰 |
| 2 | **집 — 뷰 B (책상/현관 쪽)** | ★★★☆ | 식사, 인터넷 검색, 제물 놓기, 귀신 목격. 일상 씬의 메인 뷰 |
| 3 | **집 — 뷰 C (전체/창문 쪽)** | ★★☆ | 이사, 청소, 귀신 호의, 유품 발견. 낮 씬·분위기 전환용 |
| 4 | **회사** | ★★☆ | 출근, 동료 대화, 팜플렛, 출장 |
| 5 | **거리/상점** | ★☆☆ | 쇼핑, 출퇴근 이동 |
| 6 | **사고 현장** | ★☆☆ | 노녀 구출, 남편 귀신 성불 (4막 핵심) |
| 7 | **인터넷 화면** | ★★☆ | UI 오버레이 (검색, 상담, 뉴스) |
| 8 | **VR 공간** | ★★☆ | 가상 공간 (VR 자위 내부) |
| 9 | **과거의 집 (회상)** | ★☆☆ | 초야 준비, 방화 장면. 시댁 회상은 VN 연출로 처리 |

> **실제 배경 6개** (집 3뷰 + 회사 + 거리 + 사고현장) + UI 1개 + 가상 1개 + 회상 1개 = **총 9개**
> 시댁 회상(시어머니 대립, 야반도주)은 대사·연출로 처리 — 전용 배경 불필요.
